# 数据传输

> 本文档说明如何在客户端和服务器之间传递数据

## 数据传递

### 全局数据传递

```squirrel
sq_BinaryStartWrite();
sq_BinaryWriteBool(value);      // 写入布尔值
sq_BinaryWriteFloat(value);     // 写入浮点数
sq_BinaryWriteWord(value);      // 写入字（2字节）
sq_BinaryWriteByte(value);      // 写入字节
sq_BinaryWriteDword(value);     // 写入双字（4字节）
```

### 对象数据传递

```squirrel
obj.sq_StartWrite();
obj.sq_WriteBool(value);        // 写入布尔值
obj.sq_WriteFloat(value);       // 写入浮点数
obj.sq_WriteWord(value);        // 写入字
obj.sq_WriteByte(value);        // 写入字节
obj.sq_WriteDword(value);       // 写入双字
```

### 发送数据

```squirrel
sq_SendChangeSkillEffectPacket(obj, SKILL_ID);  // 发送技能效果包
```

---

## 数据接收

### 接收数据

```squirrel
function onChangeSkillEffect_xxx(obj, skillIndex, reciveData)
{
    reciveData.readBool();      // 读取布尔值
    reciveData.readByte();      // 读取字节
    reciveData.readWord();      // 读取字
    reciveData.readDword();     // 读取双字
    reciveData.readFloat();     // 读取浮点数
}
```

## 状态数据传递

### 获取传递过来的数据

```squirrel
sq_GetVectorData(datas, index);  // 获取向量数据中的指定索引
```

### 设置状态数据

```squirrel
obj.sq_IntVectClear();           // 清空整数向量
obj.sq_IntVectPush(value);       // 推送整数到向量
obj.sq_AddSetStatePacket(state, priority, isResetTimer);  // 添加状态包
```

## 创建被动对象时传递数据

### 发送创建被动对象数据包

```squirrel
obj.sq_StartWrite();
obj.sq_WriteDword(data1);
obj.sq_WriteDword(data2);
// ... 写入更多数据
obj.sq_SendCreatePassiveObjectPacket(
    passiveObjectId,  // 被动对象ID
    subType,          // 子类型
    x, y, z,          // 坐标
    direction         // 方向
);
```

### 接收创建对象时的数据

```squirrel
function setCustomData_po_xxx(obj, receiveData)
{
    if(!obj) return;
    
    // 读取传递过来的数据
    local data1 = receiveData.readDword();
    local data2 = receiveData.readDword();
}
```

## 完整示例

### 发送示例

```squirrel
if(obj.isMyControlObject())
{
    sq_BinaryStartWrite();
    sq_BinaryWriteDword(skillId);
    sq_BinaryWriteDword(damage);
    sq_BinaryWriteFloat(rate);
    sq_SendChangeSkillEffectPacket(obj, SKILL_ID);
}
```

### 接收示例

```squirrel
function onChangeSkillEffect_xxx(obj, skillIndex, reciveData)
{
    if(!obj) return;
    if(skillIndex != SKILL_ID) return;
    
    if(reciveData.getSize() > 0)
    {
        local skillId = reciveData.readDword();
        local damage = reciveData.readDword();
        local rate = reciveData.readFloat();
        
        // 使用接收到的数据
    }
}
```
