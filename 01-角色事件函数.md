# 角色事件函数

> 本文档整理了所有角色相关的事件处理函数

## 被动技能相关

### ProcPassiveSkill - 被动技能设置

当角色学习被动技能时触发。

```squirrel
function ProcPassiveSkill_职业名(obj, skill_index, skill_level)
{
    if (skill_index == SKILL_ID)
    {
        if(skill_level > 0)
        {
            // 创建附加对象
            local appendage = CNSquirrelAppendage.sq_AppendAppendage(
                obj, obj, skill_index, false, 
                "路径/ap_xxx.nut", true
            );
        }
    }
    return true;
}
```

### onUseSkillPassiveSkill - 使用被动技能时

```squirrel
function onUseSkillPassiveSkill_职业名(obj, skillIndex, skillLevel)
{
    if(!obj) return;
    // 处理逻辑
}
```

## 状态设置相关

### onSetState_* - 设置状态时

设定状态时执行的函数。`obj.sq_AddSetStatePacket` 函数执行后会执行这个函数。

```squirrel
function onSetState_技能名(obj, state, datas, isResetTimer)
{
    if(state == STATE_DIE)
        obj.sq_RemoveSkillLoad(SKILL_ID);
    return 0;
}
```

### onAfterSetState_* - 设置状态后

```squirrel
function onAfterSetState_技能名(obj, state, datas, isResetTimer)
{
    // 状态设置后的处理
}
```

### onEndState_* - 状态结束时

```squirrel
function onEndState_技能名(obj, new_state)
{
    if(new_state != STATE_XXX)
    {
        // 状态结束时的清理操作
    }
}
```

### addSetStatePacket_* - 增加设置状态包时

可以驳回状态，且可以再次设置别的状态。

```squirrel
function addSetStatePacket_职业名(obj, state, datas)
{
    if (!obj) return -1;
    // 判断是否可以设置状态
    // 返回 -1 表示驳回状态
    return -1;
}
```

## 攻击相关事件

### onAttack_* - 攻击到时

攻击敌人后执行的函数，可以返回受到伤害的对象。**这是写抓取技能的基础**。

```squirrel
function onAttack_技能名(obj, damager, boundingBox)
{
    if(!obj) return;
    // damager: 受到攻击的对象
    // boundingBox: 攻击框
}
```

### onBeforeAttack_* - 攻击前

```squirrel
function onBeforeAttack_技能名(obj, damager, boundingBox)

    // 攻击前的处理

```

### onAfterAttack_* - 攻击后

```squirrel
function onAfterAttack_技能名(obj, damager, boundingBox)
{

    return 1;
}
```

## 定时事件

### onTimeEvent_* - 定时时钟事件

返回值应用：返回 `true` 运行成功，返回 `false` 运行不成功会多运行几次。

```squirrel
function onTimeEvent_技能名(obj, timeEventIndex, timeEventCount)
{
    if(!obj) return false;
    if(timeEventIndex == 0)
    {
        // 处理逻辑
        return true; // true 表示回调中断
    }
    return false;
}
```

## 技能控制相关

### flushCommandEnable_* - 刷新技能按键状态时

这里可以设置技能亮不亮。

```squirrel
function flushCommandEnable_职业名(obj)
{
    if (!obj) return S_FLOW_NORMAL;
    if (!obj.isInBattle())
    {
        sq_SetAllCommandEnable(obj, false);
        return S_FLOW_RETURN;
    }
}
```
