# 攻击系统

> 本文档说明攻击相关的函数和设置

## 攻击事件函数

### onAttack_* - 攻击到时

攻击敌人后执行的函数，可以返回受到伤害的对象。**这是写抓取技能的基础**。

```squirrel
function onAttack_技能名(obj, damager, boundingBox)
{
    if(!obj) return;
    // damager: 受到攻击的对象
    // boundingBox: 攻击框
}
```

### onBeforeAttack_* - 攻击前

```squirrel
function onBeforeAttack_技能名(obj, damager, boundingBox)
{
    if(!obj) return;
    // 攻击前的处理
}
```

### onAfterAttack_* - 攻击后

```squirrel
function onAfterAttack_技能名(obj, damager, boundingBox)
{
    if(!obj) return 0;
    // 攻击后的处理
    return 1;
}
```

## 攻击信息 (AttackInfo)

### 获取攻击信息

```squirrel
// 得到自定义攻击信息
local attackInfo = sq_GetCustomAttackInfo(obj, index);

// 得到当前攻击信息
local attackInfo = sq_GetCurrentAttackInfo(obj);

// 得到默认攻击信息
local attackInfo = sq_GetDefaultAttackInfo(obj, index);
```

### 设置攻击信息

```squirrel
// 设置当前攻击信息（角色）
obj.sq_SetCurrentAttackInfo(CUSTOM_ATTACK_ID);

// 设置当前攻击信息（特效）
sq_SetCurrentAttackInfo(obj, attackInfo);

// 从自定义索引设置攻击信息
sq_SetCurrentAttackInfoFromCustomIndex(obj, index);
```

### getDefaultAttackInfo_* - 得到设置默认攻击信息

```squirrel
function getDefaultAttackInfo_职业名(obj, index)
{
    if (!obj) return null;
    local attackInfo = obj.sq_GetDefaultAttackInfo(index);
    // 可以根据条件返回自定义攻击信息
    if (condition)
        attackInfo = sq_GetCustomAttackInfo(obj, CUSTOM_ATTACK_ID + index);
    return attackInfo;
}
```

## 2025神迹伤害类型系统

> 新的伤害机制结合了神迹因果最新字体的改版，添加了附加伤害、技攻和终伤等算法。

### 附加伤害（白字/小字）

在装备中，可以使用技能数据提升的方式增加后跳（169技能）的静态数值，从而在字体函数中进行绘制和计算。

**装备中的写法**：

```
[skill data up]
    `[all]`    169    `[dungeon type]`    `[static]`    1    `+`    30
[/skill data up]
```

**静态数据位置说明**：
- **169技能的1-4**：白色附加小字
- **169技能的5-17**：黄色附加小字
- **169技能的36**：最终伤害
- **169技能的37**：技攻伤害

### 最终伤害和技攻计算

最终伤害和技攻使用累加的方式计算。需要在 `sqr/character/state_damagerate.nut` 文件中定义 `getCurrentModuleDamageRate` 函数。

```squirrel
function getCurrentModuleDamageRate(obj)
{
    if (!obj)
        return 1.0;
    
    local rate = 1.0;
    
    // 最终伤害取后跳静态数据第36个
    local back_SkillLevel = sq_GetIntData(obj, 169, 36);
    
    // 技攻伤害取后跳静态数据第37个
    local back_SkillLevel2 = sq_GetIntData(obj, 169, 37);
    
    // 最终伤害与各职业单独的伤害提升进行乘算
    rate *= (100 + back_SkillLevel).tofloat() / 100.0;
    
    // 技能攻击时才计算技攻
    if(obj.getCurrentSkillIndex() > 0) 
    {
        rate *= (100 + back_SkillLevel2).tofloat() / 100.0;
    }
    
    return rate.tofloat();
}
```

**注意**：
- `getCurrentModuleDamageRate` 是自带函数，直接在nut文件中命名即可生效，无需调用
- 在最终伤害下已经排除附加小字伤害的累加计算
- 只有技攻伤害会计算附加的伤害

### 附加小字伤害函数

**文件路径**：`sqr/ui/damagefont/damagefont_function.nut`

```squirrel
// 附加小字伤害的函数
function applyAddHpDamage(appendage, number) {
    local obj = sq_GetCNRDObjectToSQRCharacter(sq_getMyCharacter());
    
    local addDamage = 0;
    local totalAddDamage = 0;
    local back_damage1 = obj.sq_GetIntData(169, 36);  // 最终伤害
    local sound = false;
    local subNumber = appendage.getVar("subNumber");
    
    for (local i = 1; i <= ADD_DAMAGE_COUNT; i++) {
        local data = obj.sq_GetIntData(169, i);
        local data2 = (100 + back_damage1).tofloat() / 100.0;
        
        if (data != 0) {
            addDamage = (data / 100.0 * number / data2).tointeger();
            if (addDamage == 0)
                addDamage = 1;
            totalAddDamage += addDamage;
            subNumber.setInt(i, addDamage);
            sound = true;
        }
    }
    return totalAddDamage;
}
```

### 异常伤害绘制

**文件路径**：`sqr/ui/ap/ap_monster.nut`

异常伤害（出血、中毒等）会跳伤害数字，频率可调整。

```squirrel
// 异常伤害
if(sq_IsValidActiveStatus(parentObj, ACTIVESTATUS_BLEEDING) || 
   sq_IsValidActiveStatus(parentObj, ACTIVESTATUS_POISON) || 
   sq_IsValidActiveStatus(parentObj, ACTIVESTATUS_LIGHTNING) || 
   sq_IsValidActiveStatus(parentObj, ACTIVESTATUS_BURN) || 
   sq_IsValidActiveStatus(parentObj, ACTIVESTATUS_STONE))
{
    local currTime = appendage.getTimer().Get();
    
    // 异常伤害ani显示的频率，数字越低显示的频率越高，帧数也越低
    if(currTime - appendage.getVar("atkTime").getInt(0) > 300)
    {
        NOMAL_ATTACK_FLAG = 1;
        appendage.getVar("atkTime").setInt(0, currTime);
        
        // 绘制异常伤害ani
        applyActiveDamage(appendage, newHpDamage, x, y, z);
    }
    
    // 计算异常伤害加成
    currentHp -= applyActiveDamage2(newHpDamage);
    if (currentHp < 1) {
        currentHp = -1
    }
    return currentHp;
}
```
