# 常用函数

> 本文档整理了开发中最常用的函数和方法

## 技能相关函数

### checkExecutableSkill_*

按下技能键后执行的函数，判断是否可以执行技能。

```squirrel
function checkExecutableSkill_SkillName(obj)
{
    // 判断逻辑
    return true; // 返回 true 表示可以执行
}
```

### checkCommandEnable_*

判断技能命令是否可用（技能图标是否亮起）。

```squirrel
function checkCommandEnable_SkillName(obj)
{
    // 判断逻辑
    return true; // 返回 true 表示技能可用
}
```

**提示**: `*` 替换为 skl 列表里的技能名称

---

## 状态相关函数

### onSetState_*

设定状态时执行的函数。

```squirrel
function onSetState_SkillName(obj, state, datas, isResetTimer)
{
    // 状态设置时的处理
    return 0;
}
```

**提示**: `obj.sq_AddSetStatePacket` 函数执行后会执行这个函数

### onEndCurrentAni_*

当前动画结束后执行的函数。

```squirrel
function onEndCurrentAni_SkillName(obj)
{
    // 动画结束后的处理
}
```

### onProc_*

在人物处于技能状态过程中会自动执行的函数（组队双方运行）。

```squirrel
function onProc_SkillName(obj)
{
    // 状态持续过程中的处理
}
```

**提示**: 这是很重要的函数，常用于持续效果的处理

---

## 攻击相关函数

### onAttack_*

攻击敌人后执行的函数，可以返回受到伤害的对象。

```squirrel
function onAttack_SkillName(obj, damager, boundingBox)
{
    // 攻击到敌人后的处理
    // damager: 受到攻击的对象
    // boundingBox: 攻击框
}
```

**提示**: 这是写抓取技能的基础函数

## 对象常用方法

### 技能使用

```squirrel
obj.sq_IsUseSkill(skillIndex)  // 使用技能，该函数使用后会使技能进入CD
                               // 没有这个函数技能无CD
```

### 状态设置

```squirrel
obj.sq_AddSetStatePacket(state, priority, isResetTimer)  // 设定状态函数
```

### 坐标设置

```squirrel
obj.setCurrentPos(x, y, z)  // 设定坐标函数
                            // 注意：人物直接用会卡墙，怪物就不会
obj.sq_SetfindNearLinearMovablePos(x, y, z)  // 不卡墙的人物坐标设置
```

### 速度设置

```squirrel
obj.sq_SetStaticSpeedInfo(type1, type2, value1, value2, rate1, rate2)  // 是否吃攻速
```

### 坐标获取

```squirrel
obj.getXPos()  // 获得x轴
obj.getYPos()  // 获得y轴
obj.getZPos()  // 获得z轴
```

### 时间判断

```squirrel
// 方式1: 使用 sq_timer_
obj.getVar().sq_timer_.isOnEvent(time)  // 判断时间函数
obj.getVar().sq_timer_.setParameter(interval, repeat)  // 设定时间变量
obj.getVar().sq_timer_.resetInstant(0)  // 重置，配合 setParameter 使用

// 方式2: 使用 setTimeEvent (推荐)
obj.setTimeEvent(index, time, count, isRepeat)  // 设定时间函数
// 与主函数 onTimeEvent_* 有关联
```

### 控制检测

```squirrel
obj.sq_IsMyControlObject()  // 检测是否使用过
```

### 移动控制

```squirrel
obj.sq_StopMove()  // 停止移动
```

### 方向设置

```squirrel
obj.setCurrentDirection(direction)  // 设定对象的朝向
```

### 动画获取

```squirrel
obj.sq_GetCurrentAni()              // 获取当前帧文件，获取帧数和帧时间的必要
obj.sq_GetCurrentFrameIndex()       // 获取帧数
sq_GetCurrentTime(obj)              // 获取帧的时间
```

## 全局函数

### 特效创建

```squirrel
sq_CreatePooledObject(ani, isAutoRemove)  // 生成临时特效函数
                                          // 第一个参数是ani，第二个默认true
```

### 方向获取

```squirrel
sq_GetDirection(obj)  // 参数写对象即可获得该对象的朝向
                      // 返回值为 ENUM_DIRECTION_RIGHT 或 ENUM_DIRECTION_LEFT
```

### 攻击信息

```squirrel
sq_GetCurrentAttackInfo(obj)  // 获得特定对象的攻击信息
```

### 异常状态

```squirrel
sq_SetChangeStatusIntoAttackInfo(attackInfo, type, status, prob, level, time, damage)
// 设置攻击异常状态
```

### 强控敌人

```squirrel
sq_HoldAndDelayDie(target, source, isControl, isDelayDie, isKnockBack, distance, height, direction, parentAp)
// 强控敌人
```

## 镜头移动

设置镜头移动需要额外添加一个函数：

```squirrel
// 在技能nut中必须有这个函数，上面的写法才生效
function getScrollBasisPos_SkillName(obj)
{
    return true;
}

// 使用示例
obj.sq_SetCameraScrollPosition(
    sq_GetDistancePos(obj.getXPos(), obj.getDirection(), 0), 
    obj.getYPos(), 
    0
);
```

## 字符串操作

```squirrel
local len = Value.len();                    // 字符串取长度
local son = Value.slice(i, i+1);            // 分割字符串
                                            // slice(起始下标, 结束下标)
                                            // 结束下标不包含
```

### 字符串分割示例

```squirrel
// 例如：Value字符串为"ab4"
// 当 i 分别为0、1、2时
local son0 = Value.slice(0, 1);  // "a"
local son1 = Value.slice(1, 2);  // "b"
local son2 = Value.slice(2, 3);  // "4"
```

## 动画映射

```squirrel
// sq_var.GetAnimationMap函数是用来获取对象上指定路径的ani，并指定名称
local sq_var = obj.getVar();
local sprChar = sq_var.GetAnimationMap("hud_e", "/ui/hud_creator_b_gauge.ani");

// 设置当前帧
sq_SetAnimationFrameIndex(sprChar, val);  // 第一个参数为ani名称，第二个为帧值

// 绘制当前帧
sq_drawCurrentFrame(ani, x, y, false);    // 在指定位置绘制动画贴图对象的当前帧
                                           // 参数：ani, x坐标, y坐标, 绘制效果
```
