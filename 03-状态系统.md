# 状态系统

> 本文档说明状态管理相关的函数

## 状态设置

### onSetState_* - 设置状态时

设定状态时执行的函数。`obj.sq_AddSetStatePacket` 函数执行后会执行这个函数。

```squirrel
function onSetState_技能名(obj, state, datas, isResetTimer)
{
    if(state == STATE_DIE)
        obj.sq_RemoveSkillLoad(SKILL_ID);
    return 0;
}
```

### onAfterSetState_* - 设置状态后

```squirrel
function onAfterSetState_技能名(obj, state, datas, isResetTimer)
{
    // 状态设置后的处理
}
```

### onEndState_* - 状态结束时

```squirrel
function onEndState_技能名(obj, new_state)
{
    if(new_state != STATE_XXX)
    {
        // 状态结束时的清理操作
    }
}
```

## 状态优先级

```squirrel
STATE_PRIORITY_AUTO <- 0           // 根据时间和条件自动完成的一切（攻击结束，发射->战场等）特效一般用这个
STATE_PRIORITY_USER <- 1           // 根据用户的command发布的内容（技能，攻击等）一般是使用技能的时候
STATE_PRIORITY_HALF_FORCE <- 2     // 想要离开，但都比强制要低（翻过来等）
STATE_PRIORITY_FORCE <- 3          // 被迫去做的事情（坠落，伤害，死亡，拦截等）一般给敌人发送控制状态时
STATE_PRIORITY_IGNORE_FORCE <- 4   // 强制变更无视的东西（用手术刀抓住对方等）一般是抓取敌人的时候
```

## 发送状态包

### 发送状态（不传值）

```squirrel
obj.sq_AddSetStatePacket(STATE_ID, STATE_PRIORITY_USER, false);
```

### 发送状态（传递值）

```squirrel
obj.sq_IntVectClear();              // 清空整数向量
obj.sq_IntVectPush(value1);         // 推送值1
obj.sq_IntVectPush(value2);         // 推送值2
obj.sq_AddSetStatePacket(STATE_ID, STATE_PRIORITY_USER, true);
```

**注意**: 最后参数为 `true` 表示传递数据，`false` 表示不传递数据。

### 特效中发送状态

```squirrel
local pIntVec = sq_GetGlobalIntVector();  // 得到包
sq_IntVectorClear(pIntVec);               // 清除
sq_IntVectorPush(pIntVec, value);         // 推送值
obj.sendStateOnlyPacket(STATE_ID, pIntVec);  // 发送状态包
```

## 状态拦截

### addSetStatePacket_* - 增加设置状态包时

可以驳回状态，且可以再次设置别的状态。

```squirrel
function addSetStatePacket_职业名(obj, state, datas)
{
    if (!obj) return -1;
    
    // 拦截特定状态
    if (state == STATE_JUMP_ATTACK && obj.getState() == STATE_JUMP)
    {
        obj.sq_IntVectClear();
        obj.sq_IntVectPush(4);
        obj.sq_addSetStatePacket(STATE_OTHER, STATE_PRIORITY_USER, true);
        return -1;  // 返回 -1 表示驳回原状态
    }
    
    return 0;  // 返回 0 或其他值表示允许设置状态
}
```

## 异常状态

### setActiveStatus_* - 设置异常状态时

```squirrel
function setActiveStatus_职业名(obj, activeStatus, power)
{
    if (!obj) return 0;
    
    local state = sq_GetState(obj);
    if (state == STATE_SPECIAL)
    {
        return 0;  // 返回0表示拒绝设置异常状态
    }
    
    return 1;  // 返回1表示允许设置异常状态
}
```

### 发送异常状态

```squirrel
// 直接发送异常状态包
// 参数：被附加对象，源对象，异常类型，概率(浮点数)，等级，强制false，时间
sq_sendSetActiveStatusPacket(
    targetObj, obj, ACTIVESTATUS_STUN,
    prob.tofloat(), level, false, time
);
```

### 在攻击信息中设置异常状态

```squirrel
// 设置攻击异常状态
// 参数：攻击信息，异常类型，概率，等级，时间，伤害
sq_SetChangeStatusIntoAttackInfo(
    attackInfo, 0, ACTIVESTATUS_BLEEDING,
    bleedingrate, bleedinglevel, bleedingtime, bleedingdamage
);
```

## HP/MP 相关

### sendSetHpPacket_* - 发送设置HP包

可以拦截HP变化。

```squirrel
function sendSetHpPacket_职业名(obj, hp, sendInstant)
{
    if(CNSquirrelAppendage.sq_IsAppendAppendage(obj, "路径/ap_xxx.nut"))
    {
        if(obj.getHp() < hp)
        {
            return false;  // 阻止HP增加
        }
    }
    return true;
}
```

function sendSetMpPacket_职业名(obj, mp, sendInstant)

    return true;

## 重置相关

### reset_* - 重置刷新时

死亡复活、进入下一个房间时执行。

```squirrel
function reset_职业名(obj)
{
    if(!obj) return;
    // 重置时的清理
}
```

### resetDungeonStart_* - 副本中重置状态时

城镇中也会运行，相当于进入副本回到城镇都会运行。

```squirrel
function resetDungeonStart_职业名(obj, moduleType, resetReason, isDeadTower, isResetSkillUserCount)
{
    if(!obj) return -1;
    // 重置处理
    return 1;
}
```
