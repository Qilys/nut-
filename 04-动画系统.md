# 动画系统

> 本文档说明动画获取、设置和管理的相关函数

## 获取动画函数

### 攻击动画

```squirrel
function getAttackAni_职业名(obj, index)
{
    if (!obj) return null;
    local animation = obj.sq_GetAttackAni(index);
    // 可以根据条件返回自定义动画
    if (condition)
        animation = obj.sq_GetCustomAni(CUSTOM_ANI_ID + index);
    return animation;
}
```

## 设置当前动画

### setCurrentAnimation_* - 得到设置当前 ani 的 ani

```squirrel
function setCurrentAnimation_职业名(obj, animation)
{
    if (!obj) return;
    if (!animation) return;
    
    // 可以对动画进行额外处理
    if (condition)
    {
        animation.setNeverApplyAnotherPlayersEffectAlphaRate(false);
        animation.setIsApplyAnotherPlayersEffectAlphaRate(false);
    }
}
```

## 攻击取消帧

### getAttackCancelStartFrameSize_* - 得到设置可攻击次数范围

```squirrel
function getAttackCancelStartFrameSize_职业名(obj)
{
    local maxAttackNumber = obj.sq_GetAttackCancelStartFrameSize();
    // 可以根据条件修改
    if (condition)
        maxAttackNumber = 2;
    return maxAttackNumber;
}
```

### getAttackCancelStartFrame_* - 得到设置攻击时在哪一帧按下键时可以进入下一个子状态

```squirrel
function getAttackCancelStartFrame_职业名(obj, index)
{
    if(!obj) return null;
    local frm = 0;
    
    if (condition)
    {
        switch (index)
        {
            case 0: frm = 7; break;
            case 1: frm = 5; break;
        }
    }
    else
        frm = obj.sq_GetAttackCancelStartFrame(index);
    
    return frm;
}
```

## 动画操作

### 获取当前动画

```squirrel
local currentAni = obj.sq_GetCurrentAni();           // 获取当前帧文件
local frameIndex = obj.sq_GetCurrentFrameIndex();    // 获取帧数
local currentTime = sq_GetCurrentTime(obj);          // 获取帧的时间
```

### 设置动画

```squirrel
obj.setCurrentAnimation(animation);      // 设置当前动画
obj.sq_SetCurrentAnimation(animationId); // 通过ID设置动画
sq_Rewind(animation);                    // 重播动画
```

### 设置帧

```squirrel
animation.setCurrentFrameWithChildLayer(frameIndex); // 设置当前帧（包含子层）
```

### 判断动画

```squirrel
sq_IsEnd(animation);           // 判断动画是否结束
sq_IsSameAni(ani1, ani2);      // 判断是否是相同的动画
```

## 动画映射

### GetAnimationMap

获取对象上指定路径的动画，并指定名称。

```squirrel
local sq_var = obj.getVar();
local ani = sq_var.GetAnimationMap("name", "路径/animation.ani");
// 或
local ani = obj.getVar().GetAnimationMap("name", "路径/animation.ani");
```

### 设置帧和绘制

```squirrel
sq_SetAnimationFrameIndex(ani, frameIndex);  // 设置当前帧
sq_drawCurrentFrame(ani, x, y, isFlip);      // 在指定位置绘制当前帧
```

## 图形效果

### 设置效果层

```squirrel
local rgb = sq_RGB(255, 0, 0);       // RGB颜色
local alpha = sq_ALPHA(255);         // 透明度
animation.setEffectLayer(
    true,                            // 是否启用
    GRAPHICEFFECT_LINEARDODGE,       // 效果类型
    true,                            // 是否应用到子层
    rgb,                             // 颜色
    alpha,                           // 透明度
    true,                            // 
    false                            // 
);
```

### 图形效果类型

```squirrel
GRAPHICEFFECT_NONE <- 0           // 无
GRAPHICEFFECT_DODGE <- 1          // 减淡
GRAPHICEFFECT_LINEARDODGE <- 2    // 线性减淡
GRAPHICEFFECT_DARK <- 3           // 变暗
GRAPHICEFFECT_XOR <- 4            // XOR
GRAPHICEFFECT_MONOCHROME <- 5     // 单色
GRAPHICEFFECT_SPACE_DISTORT <- 6  // 空间扭曲
```

### 获取当前帧效果

```squirrel
local frame = ani.GetCurrentFrame();              // 得到当前帧
local effect = frame.GetGraphicEffect();          // 得到图形效果
```
